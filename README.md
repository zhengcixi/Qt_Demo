# Qt_Demo
一些自己使用qt做的demo。
1、notepad
使用Qt实现的简易记事本。
包括以下菜单和功能：
（1）文件菜单
包括：新建、新窗口、打开、保存、另存为、打印、退出等功能。
（2）编辑菜单
包括：撤销、剪切、复制、粘贴、删除、查找、替换、全选等功能。
（3）格式菜单
包括：更换字体，自动换行功能。
自动换行功能还没有实现。
（4）查看菜单
实现功能：放大、缩小以及状态栏的查看。
（5）帮助菜单栏
实现功能：帮助菜单和关于记事本。
详细的介绍可参考博客：https://www.cnblogs.com/mrlayfolk/p/13167219.html
